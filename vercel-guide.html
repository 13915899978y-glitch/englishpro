EnglishPro — Vercel 永久部署指南

跟着做，3分钟搞定，会员随时都能访问

💡 为什么要部署到 Vercel？
• ✅ 永久在线：24小时不关机，会员随时可登录
• ✅ 免费：个人项目完全免费
• ✅ 自带 HTTPS：安全加密，手机可安装为 App
• ✅ 全球 CDN：国内也能快速访问
• ✅ 固定域名：链接永远不变，会员数据不会丢
1
下载部署包
先下载部署文件 deploy-vercel.zip（61KB），里面包含应用的所有文件。
📥 下载 deploy-vercel.zip
2
打开 Vercel 官网
在浏览器中打开 vercel.com，点击右上角 "Sign Up" 注册账号。

💡 可以直接用 GitHub 账号或 邮箱注册，30秒完成。
🔗 打开 Vercel 注册页
3
上传部署包
注册登录后：
① 点击首页的 "Add New..." → 选择 "Project"
② 找到 "Deploy without Git"（不用Git直接部署）选项，点击
③ 将刚才下载的 deploy-vercel.zip 直接拖拽到页面中
④ 点击 "Deploy" 按钮
📱 操作路径：Dashboard → Add New → Project → Deploy without Git → 拖拽 zip → Deploy
4
获取永久链接
部署完成后（约10秒），Vercel 会生成一个永久链接，格式类似：
https://englishpro-xxxxx.vercel.app
这就是你的永久访问地址！把这个链接发给会员，他们随时都能访问。
5
自定义域名（可选）
如果嫌默认域名不好记，可以在 Vercel 项目设置中：
Settings → Domains → 输入你想要的域名前缀
例如：english-learning.vercel.app

🎉 部署完成后

✅ 会员24小时随时可登录
✅ 链接永久不变
✅ 数据不会丢失
✅ 管理员 ClaudiaYao 可随时查看会员数据

⚠️ 重要提示
由于应用使用浏览器 localStorage 存储数据，不同域名的数据不共享。
所以请确定一个固定域名后，让所有会员都用同一个链接访问。
部署到 Vercel 后，域名就固定了，会员数据就不会再丢失了！
