# 📚 EnglishPro 英语学习助手 — 使用说明

## 🚀 快速开始（3步）

### 第1步：下载文件
找到 `EnglishPro.html` 文件（206KB），下载到手机或电脑。

### 第2步：用浏览器打开
- **手机**：用 Safari（iPhone）或 Chrome（Android）打开该文件

### 第3步：注册/登录
- 新用户：点击「注册」创建账号
- 老用户：直接输入用户名和密码登录
- 管理员：用户名 `ClaudiaYao`，密码 `admin2024`

## 📱 安装为手机 App（推荐）

### iPhone
1. 用 **Safari** 打开 EnglishPro.html
2. 点击底部 **分享按钮**（方框+向上箭头）
3. 选择 **「添加到主屏幕」**
4. 桌面出现 📚 图标，点开即可像原生 App 一样使用

### Android
1. 用 **Chrome** 打开 EnglishPro.html
2. 点击 **菜单**（三个点）
3. 选择 **「添加到主屏幕」**
4. 桌面出现 📚 图标，点开即可使用

## ❓ 常见问题

**Q: 数据会丢失吗？**
A: 不会。每个会员的学习数据保存在自己的浏览器中，下次打开自动恢复。

**Q: 换手机怎么办？**
A: 在旧手机上「用户菜单 → ☁️ 云备份」保存恢复码，新手机上「📥 云恢复」输入恢复码即可。

**Q: 不联网能用吗？**
A: 能！EnglishPro.html 是完全自包含的，所有功能离线可用（听力发音需要浏览器内置语音引擎）。

**Q: 管理员怎么看会员数据？**
A: 管理员在同一浏览器上注册的会员数据都能查看。用 `ClaudiaYao` 登录后进入「👑 管理」模块。
